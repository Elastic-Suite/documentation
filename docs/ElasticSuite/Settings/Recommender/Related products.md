Located under Store > Configuration > ElasticSuite > Recommender > Related products

## Related products

<img width="958" height="633" alt="image" src="https://github.com/user-attachments/assets/16b67e77-027b-426b-8801-ba8ae28e74d7" />

| Parameter | Default value | Description |
|:----------|:--------------|:------------|
| Maximum Number of Products in Related Products List | 6 | Defines the maximum number of items that will be displayed in the Related Products block on the frontend. |
| Show Related Products | Both Selected and Event-Based | Determines the data source for the recommendations. You can choose to display strictly manually selected products, automated event-based products (driven by user behavioral data), or a combination of both. |
| Extend selected products to target rule products | Yes | *(Applies to Adobe Commerce / Magento Enterprise editions).* When enabled, this feature complements your manually configured related products with products dynamically generated by Magento's native Target Rules [1, 2]. |

#### Event-Based Specific Configuration

| Parameter | Default value | Description |
|:----------|:--------------|:------------|
| Exclude composite (bundle, grouped, configurable) from Related Products List | No | If set to **Yes**, complex product types (such as configurable, grouped, or bundle products) will be strictly filtered out of your automated event-based recommendations, ensuring only simple products are displayed to the user. |
