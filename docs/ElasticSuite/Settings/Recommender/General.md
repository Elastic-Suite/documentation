## General

<img width="989" height="653" alt="image" src="https://github.com/user-attachments/assets/9372d4ea-bad2-4506-baee-b64173f07452" />

| Parameter | Default value | Description |
|:----------|:--------------|:------------|
| Exclude from recommendations any product already bought by the visitor | No | If set to **Yes**, any product that the current visitor has previously purchased will be automatically hidden from the recommendation blocks. <br><br>*Note: This strictly applies to automated, behavior-driven recommendations and does not affect manually selected/pinned products.* |
| Prevent defaulting to whole catalog | Yes | If a product is new or lacks sufficient behavioral data (views, searches, sales), the recommendation engine might not have enough context to find related items. <br><br>• **Yes:** Strictly prevents fallback recommendations; no products will be shown if there is insufficient data. <br>• **No:** The engine falls back to displaying generic, unconstrained products picked from your entire catalog. |
| Restrict Recommendations to Company Behavioral Data | No | *(Applicable only in B2B contexts).* When enabled, the product recommendation engine will exclusively use behavioral events (views, purchases, etc.) generated by users belonging to the exact same customer company, ensuring highly tailored B2B recommendations. |
