Located under Store > Configuration > ElasticSuite > Recommender > Upsell products

<img width="982" height="634" alt="image" src="https://github.com/user-attachments/assets/92bff662-9a83-406e-9dc7-39e6e1bb6075" />

| Parameter | Default value | Description |
|:----------|:--------------|:------------|
| Maximum Number of Products in Upsell Products List | 6 | Defines the maximum number of items that will be displayed in the Upsell Products block on the frontend (usually on the product detail page). |
| Show Upsell Products | Both Selected and Event-Based | Determines the data source for the recommendations. You can choose to display strictly manually selected products, automated event-based products (driven by user behavioral data), or a combination of both. |
| Extend selected products to target rule products | Yes | *(Applies to Adobe Commerce / Magento Enterprise editions).* When enabled, this feature complements your manually configured upsell products with products dynamically generated by Magento's native Target Rules. |
| Allow native complement of manually selected upsell products | Yes | **Yes:** Dispatches a native event that automatically adds any bundle product (which includes the current product) to the manual upsell selection. <br><br>*Note: Since this takes up display slots, it will potentially restrict the size of your automated event-based recommendations.* |

### Event-Based Specific Configuration

<img width="993" height="643" alt="image" src="https://github.com/user-attachments/assets/7ba654c5-c4dc-41f5-88e0-78bed6a7bef1" />

| Parameter | Default value | Description |
|:----------|:--------------|:------------|
| Force Higher Price for Upsell Products | Yes | **Yes:** Ensures that automated recommendations (driven by past tracked events) strictly feature products with a higher price than the base product. <br><br>*Note: This restriction has no impact on manually configured upsell products.* |
| Higher price limit for Upsells (base price %) | 0 | Defines a maximum price ceiling for your automated upsell recommendations. <br><br>• **0:** No upper limit. Any upsell product with a higher price than the base product can be recommended.<br>• **Value > 0:** Restricts the upsell products to a maximum of "current product price + X percent". For instance, a value of `50` will only display products up to 1.5x the base product's price. A value of `200` will only display products up to 3x the base product's price. |
| Use category views co-occurence for upsell | No | If enabled, the recommendation engine will leverage category views co-occurrence data when building the categories clauses for upsell recommendations. |
| Use product views co-occurence for upsell | No | If enabled, the recommendation engine will leverage product views co-occurrence data when building the related product clauses for upsell recommendations. |
