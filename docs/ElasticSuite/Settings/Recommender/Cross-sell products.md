Located under Store > Configuration > ElasticSuite > Recommender > Cross-sell products

## Cross sell products

<img width="978" height="513" alt="image" src="https://github.com/user-attachments/assets/a4a17deb-2b6f-4258-99af-d45145bd3ea0" />

| Parameter | Default value | Description |
|:----------|:--------------|:------------|
| Maximum Number of Products in Cross-Sell Products List | 3 | Defines the maximum number of items that will be displayed in the Cross-Sell Products block on the frontend (usually on the cart page). |
| Show Cross-Sell Products | Both Selected and Event-Based | Determines the data source for the recommendations. You can choose to display strictly manually selected products, automated event-based products (driven by user behavioral data), or a combination of both. |
| Extend selected products to target rule products | Yes | *(Applies to Adobe Commerce / Magento Enterprise editions).* When enabled, this feature complements your manually configured cross-sell products with products dynamically generated by Magento's native Target Rules [1]. |
| Compute Cross-Sell for | Last product added to cart only | Determines which products in the cart are used to generate cross-sell recommendations [2]. You can choose to base recommendations strictly on the last product added to the cart (legacy behavior), or extend the block to compute recommendations for all the products currently in the cart [2]. |
