## Main Settings

| Parameter | Default value | Description |
|:----------|:--------------|:------------|
| Navigation mode | Retail | Determines how the catalog is presented to the customer on the frontend.<br><br>• **Retail mode:** The customer will browse the standard web catalog by default.<br>• **Drive mode:** The customer will strictly see only the catalog of their chosen retailer. |
| Display offers on product page | Yes | When enabled, the product detail page will automatically display all available offers for that product across all stores/retailers. |

## Search Settings

| Parameter | Default value | Description |
|:----------|:--------------|:------------|
| Shown placeholder to help user | City, Zipcode, Address ... | Defines the placeholder text displayed inside the retailer search input field to guide customers on what information they can use to search. |
