## General Settings

| Parameter | Default value | Description |
|:----------|:--------------|:------------|
| Shown placeholder to help user | City, Zipcode, Address ... | Defines the placeholder text that appears inside the store locator search input field. This text guides customers on what kind of location information (such as their city, zip code, or street address) they can type in to find nearby stores. |

#### SEO

| Parameter | Default value | Description |
|:----------|:--------------|:------------|
| Store Locator URL | stores | Specifies the base URL path (slug) for the store locator page on your storefront. By default, customers will access the store locator by navigating to `yourdomain.com/stores`. |
